# PokéFacts

Welcome to PokéFactsV2!

This project is an updated version of [PokéFacts](https://github.com/hoyoungsin/PokeFacts) made with React and deployed with Netlify and uses [PokéAPI](https://pokeapi.co/) for Pokémon information.

You can access the app on https://pokefacts.netlify.app/.
Click on any Pokémon to open an info modal for the selected Pokémon. Use the slider and generate button to generate a new set of Pokémon.

*Currently all Pokémon will be generated randomly to encourage viewing of all species of the selected generation.

### Updates (11/26/2025)
* All Generations are now selectable!
* Sprites by game versions
* Expanding stat section to provide base stats visual.

### Added Features
* No repeat Pokémon
* Slider and Button for amount of Pokémon generated.
* Type(s) with corresponding colors, Egg Group(s), and Abilities with details.

### Planned Features
